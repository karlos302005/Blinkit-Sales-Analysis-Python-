📊 Blinkit Sales Analysis Dashboard (Python)

A data analysis project that explores Blinkit's sales dataset using Python, Pandas, Matplotlib, and NumPy. The project performs data cleaning, analysis, and visualization to uncover sales trends, outlet performance, item category insights, and customer preferences.

 🚀 Project Overview
 This project analyzes Blinkit's retail sales data and answers business questions such as:

- Which item categories generate the highest sales?
- What percentage of sales come from Low Fat vs Regular products?
- Which outlet tier performs the best?
- How does outlet size affect sales?
- What is the sales trend based on outlet establishment year?
- Which outlet location contributes the highest revenue?

  📈 Dashboard Visualizations
  The project includes the following visualizations:

- 🥧 Sales by Fat Content
- 📊 Total Sales by Item Type
- 📊 Outlet Tier by Item Fat Content
- 📉 Outlet Establishment Year vs Sales
- 🥧 Sales Distribution by Outlet Size
- 📊 Total Sales by Outlet Location Type

  🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / VS Code

  📊 Key Insights

- Low Fat products contribute **64.6%** of total sales.
- Regular Fat products contribute **35.4%**.
- Fruits & Vegetables is the highest-selling category.
- Snack Foods ranks second in overall sales.
- Tier 3 outlets generate the highest revenue.
- Medium-sized outlets contribute the largest share of sales.
- Older outlets (1998) show the highest overall sales performance.
